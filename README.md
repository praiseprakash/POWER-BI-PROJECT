# POWER-BI-PROJECT
1. Project Title / Headline
🌍 Global E-Commerce Sales Dashboard

A dynamic and interactive Power BI dashboard built to analyze global e-commerce business performance. It focuses on sales trends, customer insights, product performance, regional comparisons, and profitability to support smarter business decisions.

2. Short Description / Purpose

The Global E-Commerce Sales Dashboard is a visually engaging Power BI report designed to explore and compare sales data across multiple regions, products, and customers. It helps businesses track revenue, profit, growth trends, and top-performing segments through interactive visuals and KPI metrics.

3. Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Main platform for building interactive dashboards and reports.
📂 Power Query – Used for data cleaning, transformation, and shaping.
🧠 DAX (Data Analysis Expressions) – Used for calculated measures such as Profit Margin, YoY Growth, Ranking, and YTD Sales.
📝 Data Modeling – Star schema model with fact and dimension tables for efficient analysis.
📁 File Formats – .pbix for development, .png / .pptx for previews and presentation.


4. Data Source

Source: Sample Global E-Commerce transactional dataset.

The dataset contains 3 years of sales data (2022–2024) with detailed information on:

Customer purchases
Product categories
Regional sales
Order dates
Revenue and profit
Discounts and quantities sold

The data was structured into one Sales Fact Table and multiple dimension tables such as Customer, Product, Date, Region, and Category.

5. Features / Highlights
• Business Problem

E-commerce businesses generate massive sales data daily, but raw data alone makes it difficult to quickly answer questions like:

Which products generate the most revenue?
Which customers contribute the highest sales?
Which regions perform best?
Is the business growing year-over-year?
Which categories are most profitable?
• Goal of the Dashboard

To build an interactive reporting tool that:

Tracks sales, profit, and growth performance
Identifies top customers and products
Compares category and regional performance
Supports strategic business decisions through insights
